# 奶龙记账 — 项目文档

## 项目概述

- **项目名称**: 奶龙记账
- **项目目标**: 一款桌面端个人记账应用，帮助用户记录每一笔人民币消费，按分类查看和统计支出
- **支持平台**: Windows、macOS
- **货币单位**: 人民币（¥）
- **项目启动日期**: 2026-07-08

## MVP 功能范围（第一版）

### 核心功能

1. **记一笔**
   - 输入金额（人民币）
   - 选择分类（先选大类，再选小类）
   - 填写备注（可选）
   - 选择日期（默认当天）
   - 保存到本地数据库

2. **账单列表**
   - 按日期倒序展示消费记录
   - 显示金额、分类、备注、日期
   - 支持编辑已有记录
   - 支持删除记录（需二次确认）

3. **月度统计**
   - 本月总支出金额
   - 饼图展示各大类支出占比
   - 柱状图展示每日支出趋势

4. **分类管理**
   - 查看所有预设分类
   - 后续版本可扩展：自定义添加/编辑分类
